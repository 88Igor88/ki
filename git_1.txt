mkdir имя создаваемой папки
cd .. или cd имя папки для перехода в родительскую папку или для перехода в дочернюю папку
git init она создает локальный репозиторий
git status проверка статуса репозитория
git add имя файла или . для превращения файлов в отслеживаемые
git commit -m "" добавление комментария
git remote add origin сылка на репозиторий. помогает скопировать с удаленого репозитория на компьютер
git push origin имя ветки. отправляет изменения на удаленный репозиторий
